## Classic ciphers

## 01. Polibio
 - Key space: `none`

## 02. Caesar
 - Key space: `25`

## 03. Monoalphabetic
 - Key space: `26!` -> `403.291.461.126.605.635.584.000.000`

## 04. Affine
 - Key space: `26 * 12` = `312`

## 05. Porta
 - Key space: `none`

## 06. Vigenere
 - Key space: `26 ^ n`, where `n` is the length of the key
